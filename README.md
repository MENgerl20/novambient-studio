# NovaAmbient Studio — Ambient Sound & Focus Station

**NovaAmbient Studio** — это компактная интерактивная рабочая станция для концентрации, медитации и отдыха, реализованная как кроссплатформенное настольное приложение на стыке **Python (PyWebView)** и **Vanilla HTML/CSS/JavaScript (Web Audio API)**.

Приложение объединяет в себе процедурный генератор звуков природы и космического пространства с настраиваемым таймером продуктивности (Pomodoro Focus Timer) и адаптивным графическим визуализатором.

---

## 🌟 Ключевые возможности

### 1. ⛈️ Процедурный синтез окружения (DSP Synthesis Engine)
В отличие от стандартных медитационных приложений, проигрывающих тяжелые MP3/WAV файлы по кругу (что приводит к заметным швам на стыках аудиозаписей и требует много памяти), NovaAmbient Studio использует **100% математический синтез звуков в реальном времени**:
* **Rain (Дождь)**: Розовый шум (Pink Noise), пропущенный через низкочастотный фильтр (Lowpass) с медленной модуляцией громкости с помощью LFO (низкочастотного генератора) для симуляции порывов ветра и перепадов плотности воды.
* **Wind (Ветер)**: Розовый шум, пропущенный через высокорезонансный полосовой фильтр (Bandpass, Q=3.5), частота которого плавно блуждает под действием медленного генератора синусоидальных колебаний (LFO) в диапазоне 200 Гц — 570 Гц. Это создает естественный эффект «завывания» ветра.
* **Campfire (Костер)**: Смешение двух отдельных элементов:
  * *Низкочастотный гул*: броуновский шум (Brown Noise), пропущенный через фильтр низких частот (220 Гц) для имитации глухого пламени.
  * *Треск дерева*: специальный буфер, генерирующий случайные импульсы (попы) с коротким экспоненциальным затуханием, пропущенные через фильтр высоких частот (900 Гц).
* **Thunder (Гроза)**: Низкочастотный броуновский шум с узким фильтром (90 Гц), запускаемый по специальному алгоритму случайного таймера (каждые 20–45 секунд) с резкой атакой и плавным экспоненциальным затуханием (до 6 секунд), имитирующим раскаты грома. Сопровождается визуальной вспышкой интерфейса.
* **Cosmos Pad (Космический хор)**: 4-голосный синтезатор (аккорд C-major 7th: C3, G3, C4, E4), построенный на мягких треугольных осцилляторах, каждый из которых имеет индивидуальную модуляцию фильтра низких частот, пропущенный через линию задержки (Delay/Feedback) для объемного космического звучания.

### ⏱️ 2. Станция концентрации (Pomodoro Timer)
* Таймер обратного отсчета (Режимы: 25 минут Фокуса, 5 минут Короткого отдыха, 15 минут Длинного отдыха).
* Круговой прогресс-бар, реализованный на базе векторного SVG-кольца с динамическим изменением свойств `stroke-dashoffset` без нагрузки на центральный процессор.
* По окончании таймера проигрывается синтезированный чистый звон колокольчика (Focus Bell).

### 🎨 3. Световой визуализатор
* На базе HTML5 Canvas и встроенного узла анализа (`AnalyserNode`).
* В режиме проигрывания считывает звуковые частоты и рисует неоновые светящиеся волны с эффектом сглаживания и свечения.
* В режиме паузы переключается в режим расслабляющего движения светящихся космических частиц и мягкого дыхания синусоидальной волны.

### 💾 4. Сохранение пресетов атмосферы (.novambient)
Программа позволяет полностью сохранять текущие уровни микширования громкостей в файл. Нативный диалог сохранения и выбора файлов открывается напрямую из Python через API PyWebView, обеспечивая быстрый доступ к файловой системе Windows.

---

## 🛠️ Технологический стек
* **Бэкэнд**: Python 3, PyWebView (frameless Chromium-окно с drag-перетаскиванием).
* **Фронтэнд**: HTML5, Vanilla CSS3 (Glassmorphism layout), ES6 Modules.
* **Аудио**: Web Audio API (AudioContext, OscillatorNode, DelayNode, BiquadFilterNode, AnalyserNode).

---

## 🔍 Быстрый запуск

1. Установите зависимость:
   ```bash
   pip install pywebview
   ```
2. Запустите приложение:
   ```bash
   python main.py
   ```
3. Поднимайте ползунки громкости звуков, настраивайте таймер и наслаждайтесь чистым процедурным эмбиентом!
